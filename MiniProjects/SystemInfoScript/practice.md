# 1️⃣ Create Project Folder

```bash
mkdir practice-system-info
```

Move into it:

```bash
cd practice-system-info
```

---

# 2️⃣ Create Test Script

```bash
nano test.sh
```

Paste:

```bash
#!/bin/bash

echo "Testing Bash Script"
```

Save and exit.

---

# 3️⃣ Make Script Executable

```bash
chmod +x test.sh
```

---

# 4️⃣ Run Script

```bash
./test.sh
```

---

# 5️⃣ Practice System Commands Individually

## Current User

```bash
whoami
```

---

## Date

```bash
date
```

---

## Hostname

```bash
hostname
```

---

## Uptime

```bash
uptime
```

---

## Memory Usage

```bash
free -h
```

---

## Disk Usage

```bash
df -h
```

---

## IP Address

```bash
hostname -I
```

---

## CPU Info

```bash
lscpu
```
