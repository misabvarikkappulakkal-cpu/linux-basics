# System Info Script - Notes

# 🎯 Goal

Create a Bash script that displays useful system information automatically.

Instead of typing many commands manually,
the script collects everything in one place.

This is real Linux automation.

---

# 🧠 Concepts Learned

| Concept | Meaning |
|---|---|
| Bash Script | File containing Linux commands |
| chmod +x | Makes script executable |
| ./script.sh | Runs script |
| Command Output | Information returned by commands |

---

# 📦 Commands Used

## whoami

Shows current logged-in user.

```bash
whoami
```

---

## date

Displays current date and time.

```bash
date
```

---

## hostname

Shows computer hostname.

```bash
hostname
```

---

## uptime

Shows:
- How long system is running
- Current users
- Load average

```bash
uptime
```

---

## free -h

Displays RAM usage.

```bash
free -h
```

`-h` means human-readable.

---

## df -h

Shows disk storage usage.

```bash
df -h
```

---

## hostname -I

Displays system IP address.

```bash
hostname -I
```

---

## lscpu

Displays CPU information.

```bash
lscpu
```
