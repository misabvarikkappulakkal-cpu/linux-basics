### df

Displays disk filesystem usage.

Example:

df -h

Options:
-h = Human readable

--------------------------------

### du

Displays directory/file sizes.

Example:

du -h

Options:
-h = Human readable
--max-depth=1 = Current directory only

--------------------------------

### sort

Sort output.

sort -hr

-h = Human readable numbers
-r = Reverse order

--------------------------------

### head

Display first N lines.

head -10

--------------------------------

### find

Search files.

find . -type f

. = Current directory
-type f = Files only
