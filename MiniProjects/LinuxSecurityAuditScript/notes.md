# Linux Security Audit Script

A Bash script that performs a basic Linux security and system audit.

## Features

- Displays hostname
- Displays current user
- Shows kernel version
- Checks uptime
- Lists logged-in users
- Displays disk usage
- Displays memory usage
- Lists open ports
- Shows top running processes

## Technologies Used

- Linux
- Bash Shell Scripting

## How to Run

Make the script executable:

```bash
chmod +x audit.sh
```

Run the script:

```bash
./audit.sh
```
