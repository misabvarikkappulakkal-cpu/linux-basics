# Log Cleaner Notes

## What are Log Files?

Logs store events generated by applications and systems.

Examples:

- application.log
- error.log
- access.log
- auth.log

--------------------------------

## find Command

Search files.

Example:

find logs -type f

--------------------------------

## -mtime

File modification age.

Examples:

-mtime +7

Means:
Older than 7 days

--------------------------------

## -delete

Deletes matching files.

Example:

find logs -type f -delete

--------------------------------

## Test File Age

Create old file:

touch -d "10 days ago" file.log
