## What is Networking?

Networking allows computers to communicate and share data.

Examples:
- Browsing websites
- SSH remote access
- File transfer
- Cloud communication

---

# Important Concepts

## IP Address

Unique address assigned to a device.

Example:
192.168.x.x

---

## DNS

DNS converts domain names into IP addresses.

Example:
google.com -> IP Address

---

## Ping

Checks connectivity between devices.

Command:
ping google.com

---

## Routing

Routing decides where network packets travel.

Command:
ip route

---

## Ports

Ports identify services.

Examples:
- 22 -> SSH
- 80 -> HTTP
- 443 -> HTTPS

---

## SSH

Secure Shell for remote login.

Command:
ssh username@server-ip

---

# Useful Commands

| Command | Purpose |
|---|---|
| ip a | Show IP address |
| ping | Check connectivity |
| ss -tuln | Show open ports |
| nslookup | DNS lookup |
| curl | Download webpage |
| traceroute | Show network path |
