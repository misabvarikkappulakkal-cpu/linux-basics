# 1. Hello Script

Create:
```bash
nano hello.sh
```

Code:
```bash
#!/bin/bash

echo "Hello Linux"
```

Run:
```bash
chmod +x hello.sh
./hello.sh
```

---

# 2. Variables

Create:
```bash
nano variables.sh
```

Code:
```bash
#!/bin/bash

name="Misab"
goal="Cloud Engineer"

echo "My name is $name"
echo "I want to become a $goal"
```

# 3. User Input

Create:
```bash
nano input.sh
```

Code:
```bash
#!/bin/bash

echo "Enter your name:"
read username

echo "Welcome $username"
```
