## Basic Commands
- ls → list files
- ls -l → detailed view
- ls -a → show hidden files
- cd → change directory
- pwd → show current location

## File Operations
- mkdir folder → create folder
- touch file.txt → create file
- cp file1 file2 → copy file
- mv file1 folder/ → move file
- mv file.txt new.txt → rename file
- rm file → delete file
- rm -r folder → delete folder

## Important Rules
- Linux is case-sensitive (file.txt ≠ FILE.txt)
- No recycle bin → deletion is permanent
- mv = move + rename
