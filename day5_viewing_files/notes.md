## Commands Learned

### 1. cat
Displays the full content of a file.

Example:
cat demo.txt

---

### 2. less
Opens file in scrollable mode.

Example:
less demo.txt

Press:
- q → quit

---

### 3. more
Displays file page by page.

Example:
more demo.txt

---

### 4. head
Displays first 10 lines of a file.

Example:
head demo.txt

---

### 5. tail
Displays last 10 lines of a file.

Example:
tail demo.txt

---

- Created demo.txt file
- Added multiline content
- Viewed file using different commands
- Learned difference between head and tail
