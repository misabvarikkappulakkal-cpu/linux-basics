# Practice Commands

- pwd shows current location

- ls -l shows files with permissions

- mkdir folder_name creates a folder

- cd folder_name moves into a folder

- cd .. moves back to previous folder

- chmod 700 folder_name gives full access only to owner

- chmod 755 folder_name gives full access to owner and read+execute to others

- chmod +x file makes file executable

- touch file_name creates a new file
